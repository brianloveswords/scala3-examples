Scala 3 By Example

Hello! I'm attempting to work on an example list of snippets for introducing folks to Scala, particularly Scala 3.

I'm riffing on Go by example, so you may notice a similar progression.

Many of these examples will run in the browser, but I recommend downloading scala-cli and running them that way:

$ brew install Virtuslab/scala-cli/scala-cli
